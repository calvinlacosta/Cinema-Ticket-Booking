# Cinema Ticket Booking System

A simple PHP-based cinema booking system starter project with a modular folder structure for pages, assets, configuration, admin views, and SQL schema files.

## Getting Started
1. Place the project in a PHP-enabled web server such as XAMPP, WAMP, or Laragon.
2. Import the SQL schema from `database/cinema.sql`.
3. Open `index.php` in your browser.

## Structure
- `pages/` contains public-facing pages.
- `admin/` contains administration pages.
- `assets/` stores CSS, JavaScript, and images.
- `database/` holds SQL schema and sample data.
